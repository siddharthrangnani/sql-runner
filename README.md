# SQL Runner — FastAPI + React + SQLite3

**Assignment Submission**

This project is a Dockerized full-stack web application built with **FastAPI (backend)** and **React + Tailwind (frontend)**.  
It allows users to write and execute SQL queries directly from a browser and view the results dynamically.


##  How to Run the Project (Evaluator Instructions)

### 1️ Prerequisites
Please ensure **Docker** and **Docker Compose** are installed on your system.

### 2️ Extract the ZIP
Unzip the folder

### 3 Go into Directory
cd moengage_assignment or open the command line / bash in extracted folder

### 4 Run Docker 
docker compose up --build

### 5 Access the Application
Open the URL:-  http://localhost:3000/

For accessing SwaggerUI(backend purpose) open URL :- http://localhost:8000/docs

### 6 Stop docker containers to stop application

    docker compose down
